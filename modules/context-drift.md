# Context Drift Detection Module

Context drift occurs when communication introduces new modules prematurely.

## Purpose
To detect and correct lane shifts.

## Rules
- Identify new emotional or future-oriented modules.
- Compare against established lane.
- Flag pacing spikes.
- Rewrite using sensory anchors.

## Structure
1. Detect drift.
2. Compare to context.
3. Flag misalignment.
4. Stabilize tone.

## Example
Drift: “Right person” → premature emotional module.
Correction: “Some places hit different at the right time of year.”
