# Iteration Log Module

Tracks refinements and improvements across communication artifacts.

## Purpose
To document evolution of the OS.

## Structure
- Case observed
- Principle extracted
- Weakness identified
- Refinement applied
- SOP updated

## Example Entry
Case: Context drift detected.
Refinement: Added context-drift module.
Update: PRINCIPLES.md and METHODS.md revised.
